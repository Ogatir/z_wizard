Папка для Вани. Здесь надо будет создать проект z_wizard.gui платформы. Сюда же потом зашьем команды для комплекса и агрегатор работы с базой данных.

Разобраться с другими утилитами. Предать файл для записи резулттатов от формы к исполнителю

Правила добавления утилиты:
1) Создать в модуле gui форму с именем <название утилиты>Settings.
Создать контейнер <Имя утилиты>Params как для других утилит

2) В заголовке класса прописать реализацию интерфейса ICrossFormable. Это обязует реализовать методы

    void SaveParams(AbstractContainer container);
    void Show(AbstractContainer container);
Также прописать там тип контейнера утилиты с именем params.
3) Реализовать метод
    public Component[] GetComponents();
    Прописать в нем передачу компонентов, которые нужны для сохранения настроек

4) В методое void SaveParams(AbstractContainer container) описать сохранение параметров утилиты в Map results.
Следом results должна быть положена в контейнер утилиты.

5) В методе void Show(AbstractContainer container) необходимо полученный container привести к типу контейнера утилиты
и присвоить params его значение

6) Создать метод public void SetParams(<Тип контейнера> params).
Здесь нужно описать присвоение элементам формы значений из контейнера

7) Создать класс <Имя утилиты>Executor. Он занимается сбором параметров из контейнера в строку команды executionParams.

8) Прописать метод public void addExecutionParam(String <список имен параметров>[], <Контейнер> params).
Имена в соответствии в тем, как запаковывались в окне настроек.

9) В ExecutionManager.ExecuteUtils(UTIL_TYPE util_type, AbstractContainer paramContainers[]) прописать случай для
этой утилиты.

10) В ProjectParams прописать тип контейнера и создать для него Get/Set методы.

11) В ProjectSerializer создать метод сериализации параметрой этой утилиты.
Из массива Component[] достать элементы, привести их к нужному типу и положить паараметры в контейнер.

12) В классе MainWindow в utilSetBtn.addActionListener добавить случай запуска окна настроек этой утилиты.
Создать там контейнер параметров и реализовать метод Show().

13) В обработчике startBtn.addActionListener добавить случай для выбранной утилиты аналогично другим.
